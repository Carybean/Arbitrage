  • Click Convert: See all converted formats instantly.
  • View all results: All odds formats and implied probability are displayed for reference.

Odds Formats Explained:

  • Decimal Odds: Total return per unit staked (e.g., 2.50 = $2.50 return on $1 bet)
  • Fractional Odds: Profit per unit staked (e.g., 3/2 = $3 profit on $2 bet)
  • American Odds: Positive (+) shows profit on $100, Negative (-) shows stake needed to win $100
  • Indonesian Odds: Similar to American but uses decimals (e.g., +1.50 = +150, -4.00 = -400)
  • Hong Kong Odds: Profit per unit staked (e.g., 0.50 = $0.50 profit on $1 bet)
  • Malay Odds: Positive shows profit on 1 unit, Negative shows stake needed to win 1 unit
